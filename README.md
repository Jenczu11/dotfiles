# legendary-spork
My setup

## How to
Run chmod -R 755 on directory

Run `ohmyzsh-setup.sh`

Log out and logIn

Run `after-zsh.sh`
And change font in terminal to proper with powerline

Then run
```
basic.sh
programs.sh
npm-install.sh
```
